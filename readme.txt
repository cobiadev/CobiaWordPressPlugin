== Description ==

Helps the end-user migrate and manage their online management system including their web properties, social networking, branding and PR. All with live statistics and analytics based upon their campaigns, targets, goals as well as all other resources analyzed for them in a simple dashboard where Cobia Systems, Inc. has allowed the user to port all of this data over to their WordPress, Joomla, Drupal, Shopify, Webex and Custom installations by demand of the end-user. This allows the end-user to basically utilize WordPress as their central point control panel by hearing our client's feedback on this. A lot of requests came to simply use Cobia Systems as their central must-have in their ever changing requirements of online marketing.

Wherefore, there's only syndication of their current data from Cobia being imported into their installation as previously discussed here. In short it's a management solution to be handled on their WordPress installation by simply allowing the user to take a hold of an array of statistical data all from within their WordPress.

== Installation ==

1. Upload `Cobia` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Adjust settings
4. For detailed instructions on install, feel free to visit: https://cobiasystems.com/faq/install_wp

== Frequently asked questions ==

1. Deactivate plugin through the 'Plugins' menu in WordPress
2. Delete plugin through the 'Plugins' menu in WordPress

It's best to use the build in delete function of WordPress. That way all the stored data will be removed and no orphaned data will stay.
